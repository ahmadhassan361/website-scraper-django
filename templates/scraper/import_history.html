Import Guide
What does import do?
  • Reads website export CSV
  • Matches products by SKU (removes vendor prefixes)
  • Marks matched products as "on website"
  • Identifies new products not on website
  • Updates ProductSyncStatus for all products
Expected CSV Format:
ID,Name,Sku,Barcode,ISBN
12345,Product Name,RLC-22-N,,
12346,Another Product,IBSLA123,,

Note: The SKU column should contain the prefixed SKU as it appears on your website. The system will automatically remove the prefix to match database products.

Pro Tip: Import your website export regularly (weekly) to keep sync status accurate. This helps you identify new products that need to be added to your website.
