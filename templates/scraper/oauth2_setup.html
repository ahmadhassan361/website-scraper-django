Google OAuth2 Setup Instructions

Important: Follow these steps to enable Google Drive and Sheets integration with OAuth2 authentication.
Step 1: Create Google Cloud Project
  1. Go to the Google Cloud Console
  2. Create a new project or select an existing one
  3. Make note of your Project ID
Step 2: Enable Required APIs
  1. In the Google Cloud Console, navigate to APIs & Services > Library
  2. Search for and enable the following APIs:
    • Google Drive API
    • Google Sheets API
Step 3: Create OAuth2 Credentials
  1. Go to APIs & Services > Credentials
  2. Click Create Credentials > OAuth 2.0 Client IDs
  3. Choose Web application as the application type
  4. Set the Name (e.g., "Website Scraper OAuth2")
  5. Add Authorized redirect URIs:
    • {{ request.build_absolute_uri }}callback/
    • http://localhost:8000/scraper/google/callback/ (for local development)
  6. Click Create
Step 4: Download Credentials
  1. After creating the OAuth2 credentials, click the Download button
  2. Save the downloaded JSON file as oauth2_credentials.json
  3. Place this file in the credentials/ directory of your project
Step 5: Authorize the Application
  1. Once the credentials file is in place, click the Authorize Google Drive button
  2. You'll be redirected to Google's authorization page
  3. Sign in with your personal Google account (the one with sufficient Drive storage)
  4. Grant the requested permissions
  5. You'll be redirected back to the application
Benefits of OAuth2 Authentication:
  • Higher Storage Quota: Uses your personal Google Drive storage (15GB+ free)
  • No Storage Limits: No service account storage restrictions
  • Direct Access: Files are created in your personal Google Drive
  • Better Security: Token-based authentication with automatic refresh
  • Easy Management: View and manage exported files directly in your Drive
File Structure:

Make sure your project structure looks like this:

your-project/
├── credentials/
│   └── oauth2_credentials.json  # ← Place the downloaded file here
├── manage.py
└── ...
